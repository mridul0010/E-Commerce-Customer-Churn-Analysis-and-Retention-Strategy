# 📊 E-Commerce Customer Churn Analysis & Retention Strategy

An interactive Streamlit dashboard that analyzes customer churn patterns in e-commerce and provides actionable retention strategies backed by data-driven insights.

🔗 **[Live App](https://e-commerce-customer-churn-analysis-and-retention-strategy.streamlit.app/)**

---

## Overview

Customer churn is one of the most critical challenges facing e-commerce businesses. This project provides a comprehensive, interactive dashboard to explore churn drivers, assess revenue impact, and formulate targeted retention strategies.

The analysis covers demographics, purchasing behavior, payment methods, product categories, and spending segments to identify the highest-risk customer groups and recommend data-backed interventions.

---

## Features

- **KPI Metrics** — Total customers, revenue, churn rate, and revenue lost to churn at a glance
- **Univariate Analysis** — Distribution charts for product categories, pricing, payment methods, gender, age groups, and churn
- **Business Analysis** — Churn breakdowns by product category, age group, gender, purchase value, returns, and payment method
- **Customer Behavior** — Churn patterns by price range, spending segment, category diversity, and order quantity
- **Revenue Impact** — Quantified revenue loss due to churn across product categories
- **Retention Strategy** — Actionable recommendations including targeted campaigns, incentive programs, and personalized marketing
- **Interactive Filters** — Sidebar filters for product category, gender, age group, and payment method

## Getting Started

### Prerequisites

- Python 3.8 or higher

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/mridul0010/E-Commerce-Customer-Churn-Analysis-and-Retention-Strategy.git
   cd E-Commerce-Customer-Churn-Analysis-and-Retention-Strategy
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app**

   ```bash
   streamlit run app.py
   ```

   The dashboard will open in your browser at `http://localhost:8501`.

## Key Insights

- **High-spending customers** have a churn rate of ~34%, making them a priority retention segment.
- **Books** and **Clothing** categories account for the largest share of churn-related revenue loss.
- **Credit card** users exhibit the highest churn rate among all payment methods.
- Customer **behavior** (spending level, category diversity) is a stronger predictor of churn than demographics (gender, age).
